# Quirk ✨

A playful, fully client-side personality quiz. Answer 40 lighthearted questions and
meet your 4-letter "type" — complete with confidence bars, strengths, blind spots,
and a shareable result. Built to be **fun**, not formal: springy cards you can swipe,
a confetti reveal, and a candy-pop look.

## ⚠️ Disclaimer (please read)

## How it works

- **`src/data/questions.ts`** — 40 original Likert statements, 10 per axis
  (Energy E/I, Information S/N, Decisions T/F, Lifestyle J/P), each tagged with the
  pole a "strongly agree" answer leans toward.
- **`src/lib/scoring.ts`** — pure scoring: each answer contributes ±2 toward its pole;
  per-axis sums map to a 0–100 lean and a clarity %, then concatenate into a type.
- **`src/data/profiles.ts`** — the 16 type profiles (original nicknames & copy).
- **`src/lib/share.ts`** — compact, server-free result encoding for share links
  (`#/r/INFP.72.40.61.30`).
- **`src/lib/storage.ts`** — local history (last 50 results).
